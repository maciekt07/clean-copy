## Development

Install dependencies:

```bash
pnpm i
```

Start dev server:

```bash
pnpm dev
```

For Firefox:

```bash
pnpm dev:firefox
```

## Build

```bash
pnpm build
```

For Firefox:

```bash
pnpm build:firefox
```

## Package (ZIP)

```bash
pnpm zip
```

For Firefox:

```bash
pnpm zip:firefox
```
